HTML

שפת תגיות לתצוגה ועיצוב דפי אינטרנט ותוכן לתצוגה בדפדפן.
זו שפת התגיות המרכזית בעולם האינטרנט.
השפה מאפשרת עיצוב תוכן בצורה מהירה,
קלה ללימוד באופן יחסי וקלה לכתיבה. היא מותרת לשימוש על ידי כל מפתח אתרים,
ללא צורך ברכישת זכויות יוצרים מחברה כלשהי.
היא מאפשרת להציג נתונים בעזרת: טקסט, צליל, תמונה, וידאו ועוד.
HTML משמשת תקן ברשת האינטרנט וכלליה מוגדרים על ידי ארגון תקינה בינלאומי,
W3C. הגרסה הנוכחית של השפה היא HTML5.

Css

CSS נוצר במטרה להפריד בין תוכן ומבנה דפי האינטרנט לבין עיצובם,
נכתבו תוכן האתרים וסגנון העיצוב שלהם באותו דף HTML.
כתוצאה מכך, הפך קוד ה-HTML למסובך ובלתי קריא,
ושינויים עיצוביים באתר שלם דרשו מעבר דף אחר דף.
באמצעות CSS.
